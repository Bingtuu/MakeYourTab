# Privacy Policy for MakeYourTab

Effective Date: April 12, 2026

## Introduction

MakeYourTab is a Chrome extension designed to help users visually mark and manage open browser tabs. This privacy policy explains how the extension handles user data and privacy.

## Data Collection

MakeYourTab is a **local-first** extension. It does not collect, transmit, or store any user data on external servers.

### What Data We Store Locally

The extension may store the following data locally in the user's browser:

- **Tab markers**: User-created color, emoji, and text labels for open tabs
- **Recent tags**: History of recently used tag combinations
- **Saved presets**: User-defined tag combinations for reuse
- **Extension settings**: Language preference, floating panel position, and badge visibility

### What Data We Do Not Collect

MakeYourTab does not collect:
- Browsing history
- Personal information
- IP addresses
- Device information
- Usage analytics
- Any data that could identify individual users

## Data Storage

All data is stored using Chrome's extension storage API (`chrome.storage.local`), which is local to the user's browser and device. This data is not synchronized across devices unless the user has enabled Chrome sync for extensions.

## Permissions

MakeYourTab requests the following permissions to function:

- **tabs**: To identify, activate, and manage marked tabs
- **activeTab**: To work with the currently active tab
- **scripting**: To inject marker logic into supported pages
- **storage**: To save local extension settings and marker data
- **host permissions**: To apply and restore markers on supported pages

These permissions are used solely to support the extension's core functionality and are not used for data collection or tracking.

## Third Parties

MakeYourTab does not share user data with third parties and does not use third-party services for data processing.

## Data Retention

Local data is retained until:
- The user clears extension data
- The user uninstalls the extension
- The user manually deletes specific data

## Children's Privacy

MakeYourTab is not intended for use by children under the age of 13. We do not knowingly collect data from children.
