## 1. 隐私政策

### 1.1 核心原则

TabMarker插件严格遵守"本地化"原则，所有用户数据仅存储在用户本地浏览器，不会上传到任何外部服务器。

> **关键承诺**
> 
> - 所有数据仅存储在本地浏览器
> - 不会向任何外部服务器上传数据
> - 不会收集用户浏览历史或个人信息
> - 不会追踪用户行为或使用情况

### 1.2 数据收集说明

插件仅收集以下必要数据：

| 数据类型 | 收集方式 | 用途 |
|----------|----------|------|
| 常用Tag组合 | 用户主动输入 | 保存常用标记组合 |
| 快捷键设置 | 用户主动设置 | 存储用户定义快捷键 |
| 标签页标记 | 用户主动设置 | 标记当前标签页 |

### 1.3 数据存储位置

所有用户数据均存储在本地，具体位置如下：

| 数据类型 | 存储位置 |
|----------|----------|
| 常用Tag组合 | chrome.storage.local |
| 快捷键设置 | chrome.storage.local |
| 标签页标记 | chrome.tabs API (标签页实例) |

## 2. 权限说明

### 2.1 所需权限

TabMarker插件需要以下权限才能正常工作：

| 权限名称 | 用途说明 |
|----------|----------|
| tabs | 读取和修改标签页标题和标记 |
| storage | 本地存储用户设置 |
| activeTab | 获取当前活动标签页信息 |
| commands | 注册键盘快捷键 |

### 2.2 权限使用说明

#### tabs权限

用于读取和修改标签页的标题和标记信息。插件仅修改标签页的标题属性，不会访问页面内容。

#### storage权限

用于在本地存储用户的常用tag组合和快捷键设置。数据仅保存在用户本地浏览器。

#### activeTab权限

用于获取当前活动标签页的信息，以便快速标记。

#### commands权限

用于注册和监听键盘快捷键，提供快速操作入口。

## 3. 安全规范

### 3.1 代码安全

#### 输入验证

- 所有用户输入均需进行长度和格式验证
- 文字标签长度限制20个字符
- 禁止输入HTML和JavaScript代码
- 对特殊字符进行转义处理

#### 数据安全

- 使用JSON格式存储数据
- 对存储数据进行类型检查
- 防止数据滥用，限制存储数量

### 3.2 错误处理

- 所有异步操作均需添加try-catch处理
- 错误信息不暴露敏感内容
- 提供友好的错误提示信息

## 4. 合规声明

### 4.1 Chrome Web Store政策合规

本插件严格遵守Chrome Web Store的开发者政策：

- 使用Manifest V3标准
- 仅申请必要的权限
- 提供清晰的隐私政策说明
- 不包含恶意代码或行为

### 4.2 用户权益保障

用户拥有以下权益：

- 完全掌控自己的数据
- 随时查看和删除已保存的数据
- 卸载插件后数据自动清除
- 不受任何广告或推广干扰

> **重要声明**
> 
> TabMarker是一款纯本地工具型插件，不涉及任何网络通信。
> 用户的所有数据均存储在本地浏览器，插件开发者无法访问。
