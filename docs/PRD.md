# MakeYourTab — Product Requirements Document

> 版本: 1.0.0  
> 状态: 已发布，维护阶段  
> 目标读者: 产品经理、开发者、AI 协作者

---

## 1. 产品概述

### 1.1 一句话描述

MakeYourTab 是一个 Chrome 浏览器扩展，让用户通过颜色、Emoji 和文字标签为当前标签页添加视觉标记，从而在海量标签中快速识别、组织和切换。

### 1.2 目标用户

- 同时打开 10+ 标签页的开发者（文档、GitHub、API 参考）
- 需要对比多来源的研究人员
- 操作多个仪表盘和报表的运营人员
- 任何希望提升标签页识别效率的浏览器重度用户

### 1.3 核心价值主张

| 痛点 | 解决方案 |
|------|----------|
| 标签页多了之后难以一眼找到目标页 | 通过颜色 + Emoji + 文字前缀重构标题和图标 |
| 同类页面（如多个文档页）难以区分 | 自定义标签组合 + 预设快速复用 |
| 切换标签需要逐个查看 | 浮动面板一键跳转 + 双击 Ping 提醒 |
| 标记操作繁琐 | 最近使用 + 预设 = 一键应用 |

### 1.4 设计约束

- **不能修改 Chrome 原生标签页背景色**（Chrome Extension API 不支持）
- **不能注入 Chrome 内置页面**（chrome://*、Web Store 等）
- **数据必须完全本地存储**，不上传任何服务器
- **零外部依赖**，单文件即可运行

## 2. 用户故事

### US-1: 标记当前标签页
> 作为用户，我想给当前正在查看的标签页加个标记，以便在标签栏中快速识别它。

**验收标准:**
- 可以从 12 种预设颜色中选择
- 可以从 12 个常用 Emoji 中选择
- 可以输入最多 12 个字符的文字标签
- 三种标记元素可以单独或组合使用
- 应用后，标签页标题前缀实时变化（如 `[📘 DOC] API Reference`）
- 应用后，favicon 被替换为带颜色 + 符号的生成图标

### US-2: 快速复用最近标记
> 作为用户，我刚给 A 页面打了标记，现在想给 B 页面打同样的标记。

**验收标准:**
- 最近使用列表最多保留 6 条不重复记录
- 点击即可一键应用到当前标签页
- 列表按使用时间倒序排列

### US-3: 保存常用预设
> 作为用户，我有几套常用的标记组合（如工作文档、个人阅读、待办事项）。

**验收标准:**
- 最多保存 5 个预设
- 预设包含名称 + 颜色 + Emoji + 文字
- 预设支持拖拽排序
- 预设可在 popup 和设置页中删除

### US-4: 在当前窗口的标记标签间切换
> 作为用户，我在当前窗口打开了多个已标记的标签页，想在它们之间快速跳转。

**验收标准:**
- 每个已标记页面的右上角显示浮动面板
- 面板列出当前窗口最多 15 个已标记标签
- 点击面板中的某一项，直接激活对应标签页
- 双击某一项，触发对应标签页的 attention 动画（不切换）
- 面板可拖拽到任意位置，位置记忆到 storage

### US-5: 切换界面语言
> 作为用户，我希望界面支持中文和英文。

**验收标准:**
- Popup 和 Options 页面均支持语言切换
- 浮动面板提示文案跟随语言设置
- 切换后所有用户可见文本即时更新
- 语言偏好持久化存储

### US-6: 快捷键操作
> 作为用户，我希望不打开 popup 也能执行常用操作。

**验收标准:**
- `Alt+T` 打开 popup（默认）
- `Alt+Shift+C` 清除当前标签标记（默认）
- 支持 3 个预设快捷键槽位（需用户到 chrome://extensions/shortcuts 自行绑定）

## 3. 功能规格

### 3.1 Popup 弹窗

| 模块 | 功能 | 优先级 |
|------|------|--------|
| 头部 | 显示当前标签标题 + 域名，受限页面提示 | P0 |
| 预览区 | 实时显示标记后的标题效果 | P0 |
| 编辑器 | 颜色选择（预设网格 + color picker）| P0 |
| 编辑器 | Emoji 选择（12 宫格，点击选中/取消）| P0 |
| 编辑器 | 文字输入（max 12，实时字符计数）| P0 |
| 操作 | 应用标记 / 清除标记 | P0 |
| 最近使用 | 列表展示，点击一键应用 | P1 |
| 预设 | 保存当前组合 / 应用 / 编辑 / 删除 / 拖拽排序 | P1 |
| 快捷键 | 显示当前快捷键配置 | P2 |
| 全局 | 语言切换按钮 | P1 |

### 3.2 浮动面板（Content Overlay）

| 功能 | 规格 |
|------|------|
| 显示条件 | 页面已被标记且 `showPageBadge` 为 true |
| 位置 | 默认右上角，可拖拽，位置持久化 |
| 尺寸 | 宽度 280px，自适应高度 |
| 内容 | 头部（标记标签 + 操作提示）+ 标签列表（最多 15 项）|
| 样式 | 毛玻璃暗色面板，z-index 2147483647 |
| 列表项 | 标记标签 + 页面标题（截断）+ 颜色圆点 |
| 点击 | 220ms 防抖后激活对应标签 |
| 双击 | 清除点击 timer，发送 ping 指令；目标 tab 标题和 favicon 闪烁，但不跳转 |
| 拖拽 | pointer 事件，边界限制在视口内 8px |

### 3.3 Options 设置页

| 功能 | 规格 |
|------|------|
| 显示页面角标 | toggle，控制所有已标记页面的浮动面板显隐 |
| 语言选择 | dropdown，zh-CN / en |
| 快捷键入口 | 按钮打开 `chrome://extensions/shortcuts` |
| 预设管理 | 只读列表 + 删除按钮（低频操作入口）|
| 受限页面说明 | 静态说明文本 |

### 3.4 标记持久化与恢复

| 场景 | 行为 |
|------|------|
| 页面刷新 | 标记自动恢复（标题、favicon、浮动面板）|
| 标签页关闭 | 自动清理对应 storage 记录 |
| Service Worker 重启 | 通过 `CONTENT_READY` 机制重新建立连接 |
| 受限页面 | 只保存 storage 状态，不注入视觉变更 |

## 4. 非功能需求

### 4.1 性能

- Popup 打开到可交互 < 300ms
- 标记应用到页面标题 < 100ms
- 浮动面板渲染 < 50ms
- Storage 写操作异步，不阻塞 UI

### 4.2 兼容性

- Chrome 88+（Manifest V3）
- 支持 `https://`、`http://`、`file://` 页面（file 需用户开启权限）
- 不支持 `chrome://`、`edge://`、Chrome Web Store、部分安全限制页面

### 4.3 安全与隐私

- 所有数据存储在 `chrome.storage.local`
- 不向任何远程服务器发送数据
- `<all_urls>` 权限仅用于内容脚本注入和标记应用
